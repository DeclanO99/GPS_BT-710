# GPS_BT-710
GPS and Bluetooth module for Kenwood D710 radios


# Version 1.0

- Everything works but requires a custom cross over cable. Wiring diagram to follow
- Requires that the Power and Data lines on the GPS JST connector are swapped

# Version 1.1

- Fixed the 8p8c connections for pulling ground and power off
- Added power and Ground connection points to be powered from other source than D710.
